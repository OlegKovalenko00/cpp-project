# Сервис агрегации (Aggregation Service)

Сервис агрегации извлекает данные из таблицы `metrics`, выполняет агрегацию (среднее, максимум, минимум) и записывает агрегированные данные в таблицу `aggregated_metrics`.

## Функциональность
- Чтение сырых метрик из БД.
- Агрегация данных (AVG, MAX, MIN).
- Сохранение агрегированных данных.
