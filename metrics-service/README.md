# Сервис метрик (Metrics Service)

Сервис метрик отвечает за сбор и хранение данных о метках (например, время отклика, количество запросов). Он подключается к базе данных PostgreSQL и сохраняет метрики в таблице `metrics`.

## Функциональность
- Сбор метрик.
- Сохранение метрик в БД.

# metrics-service
